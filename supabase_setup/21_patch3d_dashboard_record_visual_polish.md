# Patch 3D — Dashboard / Record Visual Polish

Patch 3D is a visual and balance pass for the Dashboard / The Record route, built on Patch 3C.

## Scope

- Refined the Record header hierarchy with a short institutional brief.
- Improved the Index/filter control spacing, frame, and menu treatment.
- Added a restrained register-stage treatment to reduce the empty floating feeling.
- Reworked the empty Record state into a quieter framed archive state.
- Improved post spacing, left register rule, title/body hierarchy, and attachment thumbnail presence.
- Centred and refined the floating New Record composer to avoid awkward collision with the Index control.
- Improved composer field proportions, attachment preview sizing, and mobile bottom-sheet behaviour.
- Reworded category display labels in the frontend to feel more institutional:
  - Meetings → Proceedings
  - Ideas → Studies
  - Tasks → Directives
  - Product → Materials
  - Brand → House
  - Finance → Accounts
- Added Portuguese equivalents for the adjusted frontend labels.

## Not changed

- No SQL changes.
- No RLS changes.
- No Supabase Storage policy changes.
- No authentication changes.
- No Record table/schema changes.
- No Desk database changes.
- No attachment upload logic changes.
- No new chat-like features.
