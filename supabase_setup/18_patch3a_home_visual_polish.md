# Patch 3A — Home Visual Polish

This patch is part 3A of the visual optimisation sequence following Patch 15B.

Scope:

- Refined the Home composition without changing the route structure or adding new product features.
- Increased the Home monogram presence slightly while preserving the quiet, private atmosphere.
- Added a restrained Home-only centre veil, fine horizontal axis, and subtle ROMA inscription beneath the monogram.
- Added a delicate internal rule to the side navigation panel.
- Added clearer active-state styling for the current navigation route.
- Left Gallery, Divisions, Dashboard/Record, Supabase Auth, RLS, Storage, and SQL untouched.

Build marker:

```text
patch3a-home-visual-polish
```

Manual checks after deployment:

- Home loads with the monogram visible but still understated.
- Header/footer remain stable.
- Opening and closing the navigation panel remains smooth.
- Active navigation item is legible without feeling loud.
- Mobile Home keeps the monogram centred and does not collide with the header or footer.
